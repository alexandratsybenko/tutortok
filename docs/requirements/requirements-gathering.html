2. Составление списка вопросов для интервью​

Функционал: Просмотр и оценка видеовизиток в ленте

Ученики (пользователи платформы):

  1. Расскажите о вашем текущем процессе выбора репетитора на других платформах. Как вы делаете итоговый выбор? На что обращаете внимание? (Цель: сравнить функционал с конкурентами; определить критерии выбора репетитора; понять, какие боли можем закрыть)

  2. С какими проблемами вы сталкиваетесь при поиске репетитора? Приведите пример. (Цель: выяснить существующие проблемы для поиска решения с помощью нашей платформы)

  3. Что для вас самое важное в презентации репетитора? (Опыт, уровень языка, акцент, эмоциональность и др.) (Цель: выявить разделение по фильтрам; понять, какие тэги показывать в описание к визитке; разработать советы для репетиторов по привлечению внимания)

  4. Какие ожидания у вас есть от ленты видеовизиток репетиторов? (Цель: выявить первичные ожидания пользователей от нового формата)

  5. Какие элементы экрана и фильтры вы ожидаете увидеть? Что было бы полезно использовать? (Цель: описать полный UI, удовлетворяющий потребности пользователя)

  6. Где вы ожидаете увидеть ленту с видео? Это должен быть первый экран или отдельный раздел? (Цель: понять, какое расположение ленты с видео будет более интуитивным  и удобным для пользователей)

  7. Что ожидаете увидеть при клике на аккаунт репетитора? Какая информация для вас наиболее важна? (Цель: разработать оптимальное наполнение страницы репетитора)

  8. За какое время должна загружаться видеовизитка?  (Цель: выяснить время, которое клиент готов подождать. По истечении времени ожидания клиент, вероятнее всего, уйдет)

Репетиторы  (пользователи платформы):

  1. За какое время вы бы хотели презентовать себя в видеовизитке? (Цель: понять длительность видео, которые будем использовать в ленте)

  2. Важна ли для вас статистика видео? Если да, какие данные вы бы хотели получать? (Цель: обеспечить возможность репетиторам оценивать эффективность видео для привлечения учеников)

  3. Сколько времени для модерации видеовизитки для вас приемлемо? (Цель: обеспечить своевременную модерацию; избежать недовольства репетиторов из-за длительной проверки видео)

  4. Как вы хотите узнавать о лайках видео? Насколько быстро? (Цель: определить, критична ли функция уведомлений для мотивации репетиторов; понять необходимость реализации функции)

 Product Owner 

  1. По какому приоритету показываются видео, если пользователь не настроил фильтры? (Цель: определить бизнес-логику ранжирования видео для максимальной заинтересованности пользователей)

  2. Что если отсутствуют видео, прошедшие модерацию? (Цель: определить, что показывать пользователю при отсутствии контента; как мотивировать репетиторов загружать видео)

  3. На какие метрики вы обращаете внимание при оценке эффективности функционала? 

  4. Что из функционала ленты обязательно должно быть в версии 1.0?

Backend-разработчик:

  1. Какие ограничения по размеру и формату видео? Какое минимально допустимое разрешение видео? 

  2. Сколько одновременных пользователей должна выдерживать лента?
